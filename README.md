> **"Manage projects. Sync your team."**
> A modern, simple, and powerful project management platform designed for teams to organize tasks, track progress, and collaborate in real-time.

---

## ✨ Key Features

- 📋 **Project Boards**: Clean Kanban-style visualization to manage workflows from *To-Do* to *Done*.
- 👥 **Team Directory**: Manage members, assign roles, and track individual contributions effortlessly.
- 🔔 **Live Alerts**: Stay updated with real-time notifications for task assignments and project updates.
- 📊 **Dashboard**: A centralized view of your project's health, metrics, and upcoming deadlines.

## 🚀 Getting Started

### 1. Backend Setup
1. Navigate to the `/backend` folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file and add your `DATABASE_URL` and `PORT`.
4. Start the server:
   ```bash
   npm run dev
   ```

### 2. Frontend Setup
1. Navigate to the `/frontend` folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
